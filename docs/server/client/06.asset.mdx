}

### 5. 获取某个资产在指定时间的价格

```js
client.readAssetNetworkTicker('c6d0c728-2624-429b-8e0d-d9d19b6592fa', new Date('2018-01-01').toISOString());
```

返回值

```json
{
  "type": "ticker",
  "price_btc": "1",
  "price_usd": "24355"
}
```
