实例初始化

提示
  1. Mixin Messenger 为每个用户提供了两个免费的机器人创建机会。
  2. 如果没有机器人，请 点击这里 前往创建。
  3. 有疑问可以 点击这里 查看视频教程。
  4. 或者有任何其他问题可以 点击这里 进行创建问题提问。

1. 使用 keystore 创建机器人。#

const { Client } = require('mixin-node-sdk');const client = new Client({  client_id: '',  session_id: '',  pin_token: '',  private_key: '',  pin: '',});
提示

后续的章节，都是基于上边创建的 client。请先初始化好 client 实例。

2. 使用 token 创建机器人#

const token = 'eyxxxxx';const client = new Client(null, token);
