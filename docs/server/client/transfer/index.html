转账

提示

本篇文档有视频教程，请点击这里查看

1. 转账给指定 mixin 用户#

client.transfer({  asset_id: '965e5c6e-434c-3fa9-b780-c50f43cd955c',  opponent_id: 'e8e8cd79-cd40-4796-8c54-3a13cfe50115',  amount: '1',  trace_id: client.newUUID(),  memo: '转账测试',});
提示
  1. opponent_id 就是收款人的 user_id.
  2. amount 为转账数量，最小精度支持小数点后 8 位。
  3. trace_id 是一个 uuid
  4. memo 为转账备注，注意不要超过 200 字符。

返回值

2. 根据 trace_id 获取转账详情#

client.readTransfer('5a0eb95f-40da-4aaf-ad9c-b2513926a31d');
