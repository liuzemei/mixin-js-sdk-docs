### 8. 检测一个地址是否属于 mixin 主网

```js
readExternalAddressesCheck({
  asset: '43d61dcd-e413-450d-80b8-101d5e903357',
  destination: '0xCD458d7F11023556cC9058F729831a038Cb8Df9c',
});
```

```json
{
  "destination": "0xCD458d7F11023556cC9058F729831a038Cb8Df9c",
  "fee": "0",
  "tag": ""
}
```

:::tip 提示
如果是属于 `mixin` 主网的，那么 `fee` 就是 `0`，否则的话就会显示对应的提现手续费。
:::
